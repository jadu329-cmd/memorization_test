# 암송 점검 TEST 게임

성경 암송을 점검하는 인터랙티브 웹 애플리케이션입니다.

## 기능

- 전체 순서대로 암송 테스트
- 목차별로 섞어서 테스트
- 전체 무작위 섞기 테스트
- 실시간 정답 확인
- 틀린 부분만 확인하기
- 오답 노트 기능

## 사용 방법

1. 모드 선택 (전체 순서대로 / 목차별로 섞기 / 전체 무작위 섞기)
2. 목차 선택 (해당 모드인 경우)
3. 암송 내용을 입력하고 자동으로 정답 확인
4. 오답 노트에서 틀린 문제 다시 확인
